All source files related to the app are located in /src and /res.

/src/comp4350/boozr contains three packages:
/src/comp4350/boozr/application contains the main program and RecipeServices which will initialize the databse.
/src/comp4350/boozr/business is where all of the logic related to accessing the database is located.
/src/comp4350/boozr/presentation contains all processing related to the GUI elements of the app.

All test code for the project is located in comp4350.tests and comp4350.tests.business
